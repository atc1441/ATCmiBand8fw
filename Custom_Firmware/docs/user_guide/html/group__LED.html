◆ AM_DEVICES_LED_POL_DIRECT_DRIVE_M

#define AM_DEVICES_LED_POL_DIRECT_DRIVE_M   0x2

LED direct drive indicator macro.

Or this in with the polarity value to use the GPIO DATA register instead of the GPIO DATA ENABLE register to directly drive an LED buffer.

◆ AM_DEVICES_LED_POL_OPEN_DRAIN

#define AM_DEVICES_LED_POL_OPEN_DRAIN   0x4

LED OD driver macro OR this in with the polarity value to use the Open Drain version of the output driver

◆ MAX_LEDS

#define MAX_LEDS   31

Define a somewhat arbitrary maximum number of LEDs.

No board is actually expected to have this many LEDs, the value is used for parameter validation.
