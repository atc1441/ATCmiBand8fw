◆ am_hal_tpiu_enable()

void am_hal_tpiu_enable ( uint32_t  ui32SetItmBaud)

Enables the TPIU.

This function enables the ARM TPIU by setting the TPIU registers and then enabling the TPIU clock source in MCU control register.

Parameters
ui32SetItmBaud- structure for configuration. But for simplicity, ui32SetItmBaud can be set to one of the following, in which case all other structure members are ignored. In this case, the given BAUD rate is based on a div-by-8 HFRC clock.
