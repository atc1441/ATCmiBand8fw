Reads the contents of the external PSRAM into a buffer.

Parameters
pHandle- Pointer to driver handle
pui8RxBuffer- Buffer to store the received data from the PSRAM
ui32ReadAddress- Address of desired data in external PSRAM
ui32NumBytes- Number of bytes to read from external PSRAM
bWaitForCompletion- Wait for transaction completion before exiting

This function reads the external PSRAM at the provided address and stores the received data into the provided buffer location. This function will only store ui32NumBytes worth of data. If the bWaitForCompletion is true, then the function will poll for DMA completion indication flag before returning.

Returns
32-bit status

bool  bWaitForCompletion 
)

Programs the given range of psram addresses.

Parameters
pHandle- Pointer to Device driver handle
pui8TxBuffer- Buffer to write the external psram data from
ui32WriteAddress- Address to write to in the external psram
ui32NumBytes- Number of bytes to write to the external psram
bWaitForCompletion= blocking when true

This function uses the data in the provided pui8TxBuffer and copies it to the external psram at the address given by ui32WriteAddress. It will copy exactly ui32NumBytes of data from the original pui8TxBuffer pointer. The user is responsible for ensuring that they do not overflow the target psram memory or underflow the pui8TxBuffer array

Returns
32-bit status
