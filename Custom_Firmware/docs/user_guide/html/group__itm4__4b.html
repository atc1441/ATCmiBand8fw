◆ am_hal_itm_trace_port_disable()

void am_hal_itm_trace_port_disable ( uint8_t  ui8portNum)

Disable tracing on the given ITM stimulus port.

Parameters
ui8portNum

Disables tracing on the ports referred to by ui8portNum by writing the associated bit in the Trace Privilege Register in the ITM. The value for ui8portNum should be the logical OR one or more of the following values:

ITM_PRIVMASK_0_7 - disable ports 0 through 7 ITM_PRIVMASK_8_15 - disable ports 8 through 15 ITM_PRIVMASK_16_23 - disable ports 16 through 23 ITM_PRIVMASK_24_31 - disable ports 24 through 31

◆ am_hal_itm_trace_port_enable()

void am_hal_itm_trace_port_enable ( uint8_t  ui8portNum)

Enables tracing on a given set of ITM ports.

Parameters
ui8portNum- Set ports to be enabled

Enables tracing on the ports referred to by ui8portNum by writing the associated bit in the Trace Privilege Register in the ITM. The value for ui8portNum should be the logical OR one or more of the following values:

ITM_PRIVMASK_0_7 - enable ports 0 through 7 ITM_PRIVMASK_8_15 - enable ports 8 through 15 ITM_PRIVMASK_16_23 - enable ports 16 through 23 ITM_PRIVMASK_24_31 - enable ports 24 through 31
