uint32_t am_hal_mpu_global_configure ( bool  bMPUEnable,
bool  bPrivilegedDefault,
bool  bFaultNMIProtect 
)

Sets the global configuration of the MPU.

Parameters
bMPUEnable- Enable the MPU
bPrivilegedDefault- Enable the default priveleged memory map
bFaultNMIProtect- Enable the MPU during fault handlers

This function is a wrapper for the MPU_CTRL register, which controls the global configuration of the MPU. This function can enable or disable the MPU overall with the bMPUEnable parameter, and also controls how fault handlers, NMI service routines, and privileged-mode execution is handled by the MPU.

Setting bPrivilegedDefault will enable the default memory map for privileged accesses. If the MPU is enabled with this value set, only privileged code can execute from the system address map

Setting bFaultNMIProtect leaves the MPU active during the execution of NMI and Hard Fault handlers. Clearing this value will disable the MPU during these procedures.

Returns
standard hal status
