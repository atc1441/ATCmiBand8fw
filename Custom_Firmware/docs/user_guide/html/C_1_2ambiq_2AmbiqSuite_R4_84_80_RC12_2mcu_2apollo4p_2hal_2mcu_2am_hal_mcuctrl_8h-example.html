Apply various specific commands/controls on the MCUCTRL module.

Apply various specific commands/controls on the MCUCTRL module.This function is used to apply various controls to MCUCTRL.

Parameters
eControl- One of the following: AM_HAL_MCUCTRL_CONTROL_EXTCLK32K_ENABLE, AM_HAL_MCUCTRL_CONTROL_EXTCLK32K_DISABLE, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_KICK_START, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_NORMAL, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_DISABLE, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_CLOCKOUT,
pArgs- Pointer to arguments for Control Switch Case, see note and example below
Note
pArgs: new use for SDK Rev 4.4:
to use the HF XTAL clock this function now expects a pointer the following struct variable
am_hal_mcuctrl_control_arg_t
this is needed for the following eControl keywords AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_KICK_START, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_NORMAL, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_DISABLE, AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_CLOCKOUT,

am_hal_mcuctrl_control_arg_t cvar = g_amHalMcuctrlArgDefault;
// the user must choose the appropriate bit from the enum am_hal_mcuctrl_hfxtal_users_e
// there are numerous examples that use this structure
cvar.ui32_arg_hfxtal_user_mask = (1 << AM_HAL_HFXTAL_AUADC_EN);
// populate the remaing fields as needed or use the default values
retStat = am_hal_mcuctrl_control( AM_HAL_MCUCTRL_CONTROL_EXTCLK32M_NORMAL, (void *) &cvar );

Returns
status - generic or interface specific status.
