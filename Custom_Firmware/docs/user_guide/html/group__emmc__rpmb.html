uint32_t  ui32Blk,
uint32_t  ui32BlkCnt,
uint8_t *  pui8Key 
)

eMMC RPMB read function

Parameters
pCard- pointer to the card instance.
pui8RxBuffer- Buffer to store the received data from the eMMC RPMB
ui32Blk- start block number
ui32BlkCnt- read block count
pui8Key- pointer to secure key

This function reads the 'ui32BlkCnt' blocks starting from 'ui32Blk' block and saves the data in the 'pui8RxBuffer' read buffer. The caller will be blocked until all data has been sent out or failed.

Returns
status - generic or interface specific status.
