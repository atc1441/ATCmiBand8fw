◆ am_devices_mspi_ds35x1ga_read()

uint32_t am_devices_mspi_ds35x1ga_read ( void *  pHandle,
uint32_t  ui32PageNum,
uint8_t *  pui8DataBuffer,
uint32_t  ui32DataLen,
uint8_t *  pui8OobBuffer,
uint32_t  ui32OobLen,
uint8_t *  pui32EccResult 
)

Read the contents of a certain page from the NAND flash into a buffer.

Parameters
pHandle- Ds35x1ga device handle.
ui32PageNum- Page number, for this NAND, valid value from 0 to 65535.
pui8DataBuffer- Buffer to store the data read from the NAND data region.
ui32DataLen- Number of bytes to read from the NAND data region.
pui8OobBuffer- Buffer to store the data read from the NAND oob region.
ui32OobLen- Number of bytes to read from the NAND oob region.
pui32EccResult- Return ECC result according ECCS[1:0] value. 0–no bit flips; 1–bit flips corrected; 2–bit flips not corrected.

This function reads the external flash at the certain page and stores the received data into the provided buffer location. This function will only store ui32DataLen bytes of data region and ui32OobLen bytes of oob region.

Returns
32-bit status

Checks DS35X1GA timing and determine a delay setting.

Parameters
module
pDevCfg
pDevSdrCfg

This function scans through the delay settings of MSPI SDR mode and selects the best parameter to use by tuning TURNAROUND/RXNEG/RXDQSDELAY0 values. This function is only valid in SDR mode and ENABLEDQS0 = 0.

Returns
32-bit status, scan result in structure type
