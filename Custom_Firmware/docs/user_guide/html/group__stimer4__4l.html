◆ am_hal_stimer_compare_delta_set()

uint32_t am_hal_stimer_compare_delta_set ( uint32_t  ui32CmprInstance,
uint32_t  ui32Delta 
)

Set the compare value.

Parameters
ui32CmprInstanceis the compare register instance number (0-7).
ui32Deltais the value to add to the STimer counter and load into the comparator register.
Note
There is no way to set an absolute value into a comparator register. Only deltas added to the STimer counter can be written to the compare registers. Back to back writes to COMPARE are not reliable. This function takes care of adding necessary wait till the value can be safely written. If blocking in this function is not desirable, application should check for it beforehand using am_hal_stimer_check_compare_delta_set()
Returns
AM_HAL_STATUS_SUCCESS on success. The minimum value that can be set is AM_HAL_STIMER_MIN_DELTA. If the value supplied is less - the function bumps it up to safe value and returns AM_HAL_STIMER_DELTA_TOO_SMALL

◆ am_hal_stimer_int_enable()

void am_hal_stimer_int_enable ( uint32_t  ui32Interrupt)

Enables the selected system timer interrupt.

Parameters
ui32Interruptis the interrupt to be used.

This function will enable the selected interrupts in the STIMER interrupt enable register. In order to receive an interrupt from an stimer component, you will need to enable the interrupt for that component in this main register, as well as in the stimer configuration register (accessible though am_hal_stimer_config()), and in the NVIC.
