Detailed Description

These data keeps track of number of users of the HF XTAL CLOCK and the HFXTAL padout (HFXTAL clockout) when these values are non-zero the respective settings (HFXTAL clock, and HFXTAL padout, cannot be disabled) the bits in these are set for each user module according to enum am_hal_mcuctrl_hfxtal_users_e
