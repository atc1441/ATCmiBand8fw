◆ AM_HAL_PWRCTL_CRYPTO_WA

#define AM_HAL_PWRCTL_CRYPTO_WA

AM_HAL_PWRCTL_CRYPTO_WA workaround implementation uses a TIMER interrupt AM_HAL_WRITE_WAIT_TIMER (TIMER13 used for this in default SDK). The interrupt is configured as the highest priority (0) interrupt to prevent unintentional break out due to other interrupts. In order for this to work reliably, it is required that all the other interrupts in the system are set at a lower priority, reserving the highest priority interrupt exclusively for AmbiqSuite workaround.
