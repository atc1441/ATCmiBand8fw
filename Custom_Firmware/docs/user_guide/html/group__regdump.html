◆ am_util_regdump_print()

void am_util_regdump_print ( uint32_t  ui32PeriphMask,
uint32_t  ui32ModuleMask 
)

Register dumping for debug purposes.

This function dumps register values to the print port for debug purposes.

Parameters
ui32PeriphMask= an OR of the mask values to be printed. e.g. AM_UTIL_REGDUMP_IOM | AM_UTIL_REGDUMP_GPIO
ui32ModuleMask= A mask representing the modules (for a multi-module block such as IOM) to be dumped. Bit0 represents module 0, etc. This parameter is ignored for single-module blocks such as GPIO. Pre-defined macros can be used to generate this mask, e.g. REGDUMP_MOD0 | REGDUMP_MOD1 | REGDUMP_MOD2 or equivalently REGDUMP_MOD_MAS(0,2)
