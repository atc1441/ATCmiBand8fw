◆ AM_HAL_PDM_DMA_THRESHOLD_MIN

#define AM_HAL_PDM_DMA_THRESHOLD_MIN   16

DMA threshold minimum.

The PDM DMA works best if its threshold value is set to a multiple of 4 between 16 and 24, but it will technically allow threshold settings between 4 and 24. This macro sets the minimum threshold value that the HAL layer will allow.
