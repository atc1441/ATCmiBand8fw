RTC Alarm Repeat Interval.

Macro definitions for the RTC alarm repeat interval.

These macros correspond to the RPT bits in the RTCCTL register. They may be used with the am_hal_rtc_alarm_interval_set() function.

Note: AM_HAL_RTC_ALM_RPT_10TH and AM_HAL_RTC_ALM_RPT_100TH do not correspond to the RPT bits but are used in conjunction with setting the ALM100 bits in the ALMLOW register.
