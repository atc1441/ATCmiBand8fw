Sets the interface for printf calls.

Parameters
pfnCharPrint- Function pointer to be used to print to interface

This function initializes the global print function which is used for printf. This allows users to define their own printf interface and pass it in as a am_util_stdio_print_char_t type.

Parameters
*pcBuf- Pointer to the buffer to store the string
n- number of arguments
*pcFmt- Pointer to formater string

A lite version of vsprintf(). Currently handles the following specifiers: c s %[0][width]d (also i) %[0][width]u %[0][width]x

Note than any unrecognized or unhandled format specifier character is simply printed. For example, "%%" will print a '' character.

◆ am_util_stdio_strtoul()

uint32_t am_util_stdio_strtoul ( const char *  str,
char **  endptr,
int  base 
)

Converts strings to 32-bit unsigned integers.

Parameters
*str- Pointer to the string to convert
**endptr- strtoul will set this to the char after the converted num
base- Base of the number as written in the input string.

This function was implemented in a way similar to the strtoul function found in the C standard library. It will attempt to extract a numerical value from the input string, and return it to the caller. Invalid input strings will return a value of zero.

Returns
uint32_t representing the numerical value from the string.

bool am_util_stdio_textmode_set ( bool  bSetTextTranslationMode)

Text mode translates linefeed (
) characters to carriage return/ linefeed (CR/LF) combinations in printf() and sprintf() functions.

Parameters
bSetTextTranslationMode- true: Do LF to CR/LF translation. false: Don't do the text mode translation.

This function causes the printf() and sprintf() functions to translate newline characters (\n) into CR/LF (\r\n) combinations.

Returns
Previous mode.
