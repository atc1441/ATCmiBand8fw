DEVPWREN and DEVPWRSTATUS Mask Macros

The below DEVPWREN and DEVPWRSTATUS masks are used to check if a peripheral has been disabled properly

The original check of ((PWRCTRL->DEVPWRSTATUS & ui32PeriphStatus) == 0) will fail when more than one enable in the same domain is set and the user tries disable only one.
