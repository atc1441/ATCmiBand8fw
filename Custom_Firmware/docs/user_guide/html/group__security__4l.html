◆ am_hal_crc32()

uint32_t am_hal_crc32 ( uint32_t  ui32StartAddr,
uint32_t  ui32SizeBytes,
uint32_t *  pui32Crc 
)

Compute CRC32 for a specified payload.

Parameters
ui32StartAddr- The start address of the payload.
ui32SizeBytes- The length of payload in bytes.
pui32Crc- Pointer to variable to return the computed CRC.

This function uses the hardware engine to compute CRC32 on an arbitrary data payload. The payload can reside in any contiguous memory including external memory.

Returns
Returns AM_HAL_STATUS_SUCCESS on success
