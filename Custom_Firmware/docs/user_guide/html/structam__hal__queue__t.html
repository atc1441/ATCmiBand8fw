Detailed Description

A data structure that will operate as a queue.

This data structure holds information necessary for operating a thread-safe queue. When declaring a structure of type am_hal_queue_t, you will also need to provide some working memory for the queue to use. For more information on setting up and using the am_hal_queue_t structure, please see the documentation for am_hal_queue_init().
