TEX0BASE - Base Address 0

Address:

  Instance 0 Address:   0x40090000

Description:

Base address of the drawing surface 0 (must be word aligned).

Example Macro Usage:

//
// Register access is all performed through the standard CMSIS structure-based
// interface. This includes module-level structure definitions with members and
// bitfields corresponding to the physical registers and bitfields within each
// module. In addition, Ambiq has provided instance-level macros for modules
// that have more than one physical instance and a generic AM_REGVAL() macro
// for directly accessing memory by address.
//
// The following examples show how to use these structures and macros:

// Setting the ADC configuration register...
AM_REGVAL(0x50010000) = 0x1234;              // by address.
ADC->CFG = 0x1234;                           // by structure pointer.
ADCn(0)->CFG = 0x1234;                       // by structure pointer (with instance number).

// Changing the ADC clock...
ADCn(0)->CFG_b.CLKSEL = 0x2;                 // by raw value.
ADCn(0)->CFG_b.CLKSEL = ADC_CFG_CLKSEL_HFRC; // using an enumerated value.

Bits Name RW Description
31:24 IMGFMT RW Image Format

RGBX8888 = 0x0 - Color Space RGBX means, that the pixel format still has an alpha channel, but it is ignored, and is always set to 255. The RGBX 32 bit RGB format is stored in memory as 8 red bits, 8 green bits, 8 blue bits, and 8 ignored bits.
RGBA8888 = 0x1 - Color Space RED GREEN BLUE ALPHA (internal format is always on: 32-bit)
XRGB8888 = 0x2 - Color Space
ARGB8888 = 0x3 - Color Space In the ARGB (word-order) encoding the intensity of each channel sample is defined by 8 bits, and are arranged in memory in such manner that a single 32-bit unsigned integer has the alpha sample in the highest 8 bits, followed by the red sample, green sample and finally the blue sample in the lowest 8 bits
RGBA565 = 0x4 - Color Space RED(5-bits) GREEN(6-bits) BLUE (5-bits)
RGBA5551 = 0x5 - Color Space Red,green,blue,alpha (transparency).
L8 = 0x9 - Color Space Lum8 grayscale
TSC4 = 0x12 - Color Space Proprietary texture compression
TSC6 = 0x16 - Color Space Optional proprietary texture compression
TSC6A = 0x17 - Color Space Optional proprietary texture compression
23:16 IMGMODE RW Image Mode

POINTSAMPLE = 0x0 - Texture mapping: rotate, resize and distort a bitmap image. Nearest neighbor sampling.
BILINEARFILTERING = 0x1 - Texture mapping: rotate, resize and distort a bitmap image. A method used to smooth textures when displayed larger or smaller than they actually are.
15:0 IMGSTRD RW image stride (signed) distance in bytes from one scanline to another
